what is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, representing the structure of HTML and XML as a tree of objects. It allows developers to manipulate the document's content, structure, and styles dynamically. The DOM provides a way for scripts (like JavaScript) to interact with and update the webpage in real-time.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like document.getElementById() to target an element by its ID, document.querySelector() to select the first matching element by CSS selector, or document.getElementsByClassName() and document.getElementsByTagName() to target elements by class or tag. These methods allow you to interact with and manipulate the selected elements.

What is event delegation in the context of the DOM, and whay is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for multiple child elements. It leverages event bubbling, allowing events triggered by child elements to be handled by their ancestor. This is useful for improving performance and dynamically handling elements added later.
